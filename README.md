# 🚀 Nexus Archive - Quick Start Guide

## 30-Second Setup

1. **Open** `index.html` in any modern browser
2. **Start creating** novels, characters, and lore
3. **Your data auto-saves** to browser storage
4. **Export** to JSON files from Settings page

---

## Essential Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `ESC` | Close modal |
| `Click outside modal` | Close modal |

---

## File Export Cheat Sheet

| File | Contains | When to Export |
|------|----------|----------------|
| `novels.json` | All your novels | After adding/editing novels |
| `characters.json` | All characters | After creating characters |
| `lore.json` | World-building entries | After adding lore |
| `chapters.json` | All chapters | After writing chapters |
| `settings.json` | Preferences | After changing settings |
| `nexus-archive-backup.json` | **Everything** | **End of each session** |

## Common Tasks

### Create a Novel
1. Click **+ New** button (top right)
2. Fill in title, genre, description
3. Click **Save Novel**

### Add a Chapter
1. Click on a novel card
2. Click **Add Chapter** button
3. Enter chapter details
4. Click **Save Chapter**

### Create a Character
1. Go to **Characters** page
2. Click **New Character**
3. Fill in name, role, power level
4. Click **Save Character**

### Add Lore
1. Go to **Lore Archive** page
2. Click **New Entry**
3. Write your lore content
4. Click **Save Entry**

### Export Data
1. Go to **Settings** (gear icon)
2. Click **Download All JSON Files**
3. Save files to your backup folder

### Import Data
1. Go to **Settings**
2. Click **Import Full Backup**
3. Select your backup file
4. Data restored!

---

## Data Safety Rules

1. **Export before closing** browser
2. **Keep backups** in cloud storage
3. **Date your backups** (archive-2026-01-15.json)
4. **Never edit** JSON without validating

## Tips for New Users

1. **Start with one novel** - Don't overwhelm yourself
2. **Add characters as you go** - Link them to novels
3. **Use tags** - Makes searching easier
4. **Export regularly** - Better safe than sorry
5. **Use the search** - Find anything quickly
